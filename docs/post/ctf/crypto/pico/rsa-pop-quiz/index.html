picoCTFの過去問が，常設ctfとして楽しめるpicoGymの問題です．

概要#

Class, take your seats! It's PRIME-time for a quiz... nc jupiter.challenges.picoctf.org 18821

ncすると，RSA暗号の基本知識を問われます．

解答#

基本#

この問題を解くにあたって，次の各パラメータの関係を知っておく必要があります． $p,q$を素数，$n=pq$とする．また，$e$を適当な正の整数とする． 平文(plaintext)を$m$とすると，暗号文(ciphertext)$c$は $$c = m^e \mod n$$ で計算され，秘密鍵$d$を $$d = e^{-1} \mod \phi(n)$$ と計算すると，平文$m$が $$m = c^d \mod n$$ で求めることができる． なお，この場合，$\phi(n) = (p-1)(q-1)$である．

pythonでは，$a^b \mod c$をpow(a,b,c)で計算することができる($b=-1$でも動作する)．

解法#

1問目#

Today we will be taking a pop quiz, so I hope you studied. Cramming just will not do!

Inputs and outputs are in decimal. No hex here!

q : 60413
p : 76753
##### PRODUCE THE FOLLOWING ####
n
IS THIS POSSIBLE and FEASIBLE? (Y/N):

$p,q$の値が与えられ，$n$を求めることは可能ですか？，と聞かれています． $n=pq$の定義より，$n$は計算可能です． このような，単純な計算はインタラクティブ機能が便利です．

>>> p = 60413
>>> q = 76753
>>> n = p*q
>>> n
4636878989

$n,p$の値から$q$を求める問題です．$n=pq$から$p=n/q$で求められそうです． 一応，$n$が$p$で割り切れない場合も考慮しておきます．

>>> p = 54269
>>> n = 5051846941
>>> q = n//p
>>> q
93089
>>> p*q == n
True

$pq=n$を満たす$p$と$q$を求める問題です． RSA暗号は，$n$から$p,q$を計算することは困難である(ことを利用して安全性としている)ので，これはできません． 実際，頑張れば素因数分解できるかもしれませんが，FEASIBLEという単語が，「単純に可能だ」みたいなニュアンスなので，無理やり素因数分解するのはnot FEASIBLEということなのだと思います．
