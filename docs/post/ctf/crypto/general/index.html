Crypto.Util.number#

long_to_bytes,bytes_to_long#

bytes型とlong型を変換する．

>>> bytes_to_long(b'hello')
448378203247
>>> long_to_bytes(0x68656c6c6f)
b'hello'

inverse#

inverse(a,b)でaをmodbでの逆元を得られる．

>>> inverse(3,7)
5
>>> 3*inverse(3,7)%7
1

なお，デフォルトで使えるpow関数で代用可能．

>>> pow(3,-1,7)
5

getPrime#

getPrime(a)でaビットの素数が生成される

>>> getPrime(10)
911
>>> bin(911)
'0b1110001111'
>>>

Crypto.PublicKey#

pemファイルを開くことができる． openssl rsa -text -in idk.key

使用例

from Crypto.PublicKey import RSA
f=open('key.pem','rb')
pubkey = RSA.importKey(f.read())
print(pubkey)
print("n:",pubkey.n)
print("e:",pubkey.e)
